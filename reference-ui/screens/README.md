# UI 스크린샷 참고 폴더

이 폴더에 사용자가 제공한 UI 스크린샷을 넣어주세요.

## 파일 목록 (예시 파일명)

| 파일명          | 화면             |
|----------------|-----------------|
| Loading.png    | 로딩 화면        |
| Start.png      | 시작 화면        |
| Login.png      | 로그인 화면      |
| Main_Buy.png   | 메인 화면 (매수) |
| Main_Sell.png  | 메인 화면 (매도) |
| Trades.png     | 전적 기록 화면   |
| Holdings.png   | 보유 종목 화면   |
| Ranking.png    | 랭킹 화면        |
| Market.png     | 시장분석 화면    |

## 사용 방법

1. 위 파일명에 맞춰 스크린샷 이미지를 이 폴더(`/reference-ui/screens/`)에 복사하세요.
2. 개발 시 이 이미지를 열어 레이아웃/톤/색상을 맞추는 참고자료로 활용합니다.
3. 앱 코드에서 이 폴더의 이미지를 직접 import하지는 않습니다.
